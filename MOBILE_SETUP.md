# 📱 راهنمای نصب Jobbaazar روی موبایل

## 🚀 روش‌های استفاده روی iPhone

### 1️⃣ **روش اول: PWA (Progressive Web App) - بهترین گزینه**

#### مراحل نصب:
1. **Safari را باز کنید**
2. **آدرس سایت را وارد کنید**: `http://localhost:3000` (یا آدرس سرور)
3. **روی دکمه Share کلیک کنید** (مربع با فلش)
4. **"Add to Home Screen" را انتخاب کنید**
5. **نام اپلیکیشن را تایید کنید**
6. **"Add" را بزنید**

#### مزایای PWA:
- ✅ مثل اپلیکیشن اصلی کار می‌کند
- ✅ آفلاین کار می‌کند
- ✅ اعلان‌ها دریافت می‌کند
- ✅ روی Home Screen قرار می‌گیرد
- ✅ بدون نیاز به App Store

### 2️⃣ **روش دوم: Bookmark در Safari**

#### مراحل:
1. **Safari را باز کنید**
2. **آدرس سایت را وارد کنید**
3. **روی دکمه Share کلیک کنید**
4. **"Add Bookmark" را انتخاب کنید**
5. **نام را تایید کنید**
6. **"Save" را بزنید**

### 3️⃣ **روش سوم: استفاده مستقیم از مرورگر**

#### مراحل:
1. **هر مرورگری را باز کنید** (Safari, Chrome, Firefox)
2. **آدرس سایت را وارد کنید**
3. **مستقیم استفاده کنید**

## 🔧 راه‌اندازی سرور برای موبایل

### **برای استفاده در شبکه محلی:**

#### 1. آدرس IP کامپیوتر را پیدا کنید:
```bash
# در macOS/Linux:
ifconfig | grep "inet " | grep -v 127.0.0.1

# در Windows:
ipconfig
```

#### 2. سرورها را راه‌اندازی کنید:
```bash
# Frontend (در پوشه frontend):
npm run dev -- -H 0.0.0.0

# Backend (در پوشه backend):
npm start
```

#### 3. از موبایل به آدرس زیر دسترسی پیدا کنید:
```
Frontend: http://[IP-ADDRESS]:3000
Backend: http://[IP-ADDRESS]:5050
```

### **مثال:**
اگر IP کامپیوتر شما `192.168.1.100` باشد:
- Frontend: `http://192.168.1.100:3000`
- Backend: `http://192.168.1.100:5050`

## 🌐 راه‌اندازی برای اینترنت

### **1. Vercel (رایگان):**

#### مراحل:
1. **حساب Vercel ایجاد کنید**: https://vercel.com
2. **پروژه را push کنید**:
```bash
git add .
git commit -m "Add PWA support"
git push origin main
```
3. **Vercel را به GitHub متصل کنید**
4. **اتوماتیک deploy می‌شود**

### **2. Netlify (رایگان):**

#### مراحل:
1. **حساب Netlify ایجاد کنید**: https://netlify.com
2. **پروژه را drag & drop کنید**
3. **آدرس URL دریافت می‌کنید**

### **3. Heroku (پولی):**

#### مراحل:
1. **حساب Heroku ایجاد کنید**
2. **Heroku CLI نصب کنید**
3. **پروژه را deploy کنید**:
```bash
heroku create jobbaazar-app
git push heroku main
```
