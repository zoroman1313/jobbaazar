# 🏗️ فاز 2: بازارگاه مواد ساختمانی

## 📋 خلاصه فاز 2

بازارگاه مواد ساختمانی یک پلتفرم خرید و فروش مواد ساختمانی اضافی است که به کاربران امکان می‌دهد کالاهای خود را بفروشند و از دیگران خرید کنند.

## ✨ ویژگی‌های اصلی

### 🛍️ مدیریت محصولات
- **ثبت کالا**: فرم کامل برای ثبت کالا با عکس، توضیحات و قیمت
- **دسته‌بندی**: 10 دسته مختلف مواد ساختمانی
- **وضعیت کالا**: نو، دست دوم، عالی، خوب، متوسط، ضعیف
- **قیمت‌گذاری**: قیمت اصلی و قیمت فروش

### 🔍 جستجو و فیلتر
- **جستجوی متنی**: در عنوان، توضیحات و نام فروشنده
- **فیلتر دسته‌بندی**: انتخاب دسته خاص
- **فیلتر وضعیت**: انتخاب وضعیت کالا
- **مرتب‌سازی**: بر اساس قیمت، تاریخ و محبوبیت

### 👤 پروفایل فروشنده
- **امتیازدهی**: سیستم امتیاز 1-5 ستاره
- **نظرات**: نظرات خریداران
- **اطلاعات تماس**: تلفن و ایمیل
- **تاریخ عضویت**: مدت زمان عضویت

### ❤️ علاقه‌مندی‌ها
- **افزودن به علاقه‌مندی**: ذخیره کالاهای مورد علاقه
- **لیست علاقه‌مندی‌ها**: مشاهده کالاهای ذخیره شده
- **اشتراک‌گذاری**: اشتراک‌گذاری کالا با دیگران

## 🔧 API Endpoints

### محصولات
```
GET    /api/product              # دریافت همه محصولات
GET    /api/product/:id          # دریافت محصول خاص
POST   /api/product              # ایجاد محصول جدید
PUT    /api/product/:id          # ویرایش محصول
DELETE /api/product/:id          # حذف محصول
```

### نظرات و امتیازدهی
```
POST   /api/product/:id/reviews  # افزودن نظر
POST   /api/product/:id/favorite # تغییر وضعیت علاقه‌مندی
```

### محصولات کاربر
```
GET    /api/product/user/favorites  # علاقه‌مندی‌های کاربر
GET    /api/product/user/products   # محصولات کاربر
```

## 📊 مدل داده

### Product Schema
```javascript
{
  title: String,           // عنوان محصول
  category: String,        // دسته‌بندی
  description: String,     // توضیحات
  price: Number,          // قیمت
  originalPrice: Number,  // قیمت اصلی
  condition: String,      // وضعیت
  location: String,       // موقعیت
  seller: {               // اطلاعات فروشنده
    id: ObjectId,
    name: String,
    phone: String,
    email: String,
    rating: Number,
    reviews: Number
  },
  photos: [String],       // عکس‌ها
  rating: Number,         // امتیاز کلی
  reviews: [Review],      // نظرات
  status: String,         // وضعیت (فعال/فروخته/غیرفعال)
  views: Number,          // تعداد بازدید
  favorites: [ObjectId],  // علاقه‌مندی‌ها
  tags: [String],         // برچسب‌ها
  createdAt: Date,        // تاریخ ایجاد
  updatedAt: Date         // تاریخ به‌روزرسانی
}
```

## 🚀 نحوه استفاده

### 1. دسترسی به بازارگاه
- از صفحه پروفایل روی دکمه "بازارگاه" کلیک کنید
- یا مستقیماً به `/marketplace` بروید

### 2. فروش کالا
- روی دکمه "فروش کالا" کلیک کنید
- فرم را با اطلاعات کامل پر کنید
- عکس‌های کالا را آپلود کنید
- کالا را ثبت کنید

### 3. خرید کالا
- کالاهای مورد نظر را جستجو کنید
- فیلترهای مناسب را اعمال کنید
- روی کالا کلیک کنید تا جزئیات را ببینید
- با فروشنده تماس بگیرید
