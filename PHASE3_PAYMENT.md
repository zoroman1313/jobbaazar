# 💳 فاز 3: سیستم پرداخت درون برنامه

## 📋 خلاصه فاز 3

سیستم پرداخت درون برنامه یک کیف پول دیجیتال کامل است که امکان شارژ، برداشت، پرداخت و مدیریت تراکنش‌های مالی را فراهم می‌کند.

## ✨ ویژگی‌های اصلی

### 💰 کیف پول دیجیتال
- **موجودی**: نمایش موجودی فعلی با قابلیت مخفی کردن
- **شارژ**: افزودن پول از طریق کارت، بانک یا پی‌پال
- **برداشت**: انتقال پول به حساب بانکی
- **امنیت**: PIN 4-6 رقمی و احراز هویت دو مرحله‌ای

### 💳 پرداخت امن
- **پرداخت مستقیم**: خرید کالا با موجودی کیف پول
- **تراکنش‌های محافظت شده**: تأیید و تأمین مالی
- **رسید دیجیتال**: مستندات کامل تراکنش
- **بازپرداخت**: سیستم بازپرداخت خودکار

### 📊 مدیریت مالی
- **تاریخچه تراکنش‌ها**: گزارش کامل تمام تراکنش‌ها
- **آمار و نمودار**: تحلیل هزینه‌ها و درآمدها
- **فیلتر و جستجو**: جستجو بر اساس نوع، وضعیت و تاریخ
- **صفحه‌بندی**: نمایش تراکنش‌ها در صفحات

### ⚙️ تنظیمات پیشرفته
- **شارژ خودکار**: شارژ خودکار در صورت کمبود موجودی
- **اعلان‌ها**: اطلاع‌رسانی تراکنش‌ها و موجودی کم
- **محدودیت‌ها**: تعیین حد روزانه و ماهانه
- **امنیت**: تنظیمات PIN و احراز هویت

## 🔧 API Endpoints

### کیف پول
```
GET    /api/wallet/balance           # دریافت موجودی
GET    /api/wallet/transactions      # تاریخچه تراکنش‌ها
GET    /api/wallet/stats             # آمار کیف پول
```

### تراکنش‌ها
```
POST   /api/wallet/deposit           # شارژ کیف پول
POST   /api/wallet/withdraw          # برداشت از کیف پول
POST   /api/wallet/pay               # پرداخت برای کالا
GET    /api/wallet/transaction/:ref  # جزئیات تراکنش
```

### تنظیمات
```
PUT    /api/wallet/settings          # تغییر تنظیمات
PUT    /api/wallet/pin               # تغییر PIN
```

## 📊 مدل داده

### Wallet Schema
```javascript
{
  user: ObjectId,              // کاربر
  balance: Number,             // موجودی فعلی
  currency: String,            // واحد پول (GBP/USD/EUR)
  status: String,              // وضعیت (فعال/معلق/بسته)
  transactions: [Transaction], // تراکنش‌ها
  settings: {                  // تنظیمات
    autoRecharge: {            // شارژ خودکار
      enabled: Boolean,
      threshold: Number,
      amount: Number
    },
    notifications: {           // اعلان‌ها
      lowBalance: Boolean,
      transactionComplete: Boolean,
      paymentReceived: Boolean
    },
    limits: {                  // محدودیت‌ها
      dailySpend: Number,
      monthlySpend: Number,
      maxBalance: Number
    }
  },
  security: {                  // امنیت
    pin: String,               // PIN 4-6 رقمی
    twoFactorEnabled: Boolean, // احراز هویت دو مرحله‌ای
    lastLogin: Date,
    loginAttempts: Number,
    lockedUntil: Date
  }
}
```

### Transaction Schema
```javascript
{
  type: String,                // نوع (شارژ/برداشت/پرداخت/بازپرداخت/انتقال)
  amount: Number,              // مبلغ
  currency: String,            // واحد پول
  status: String,              // وضعیت (در انتظار/تکمیل/ناموفق/لغو)
  description: String,         // توضیحات
  reference: String,           // شماره مرجع یکتا
  relatedProduct: ObjectId,    // کالای مرتبط
  relatedUser: ObjectId,       // کاربر مرتبط
  paymentMethod: String,       // روش پرداخت
  fees: Number,                // کارمزد
  metadata: Mixed,             // اطلاعات اضافی
  createdAt: Date,             // تاریخ ایجاد
  completedAt: Date            // تاریخ تکمیل
}
```

## 🚀 نحوه استفاده

### 1. دسترسی به کیف پول
- از صفحه پروفایل روی دکمه "کیف پول" کلیک کنید
- یا مستقیماً به `/wallet` بروید

### 2. شارژ کیف پول
- روی دکمه "شارژ" کلیک کنید
- مبلغ مورد نظر را وارد کنید
- روش پرداخت را انتخاب کنید
- تراکنش را تأیید کنید

### 3. خرید کالا
- در صفحه کالا روی "پرداخت با کیف پول" کلیک کنید
- موجودی خود را بررسی کنید
- تراکنش را تأیید کنید
- رسید دریافت کنید

### 4. برداشت پول
- روی دکمه "برداشت" کلیک کنید
- مبلغ و حساب بانکی را وارد کنید
- تراکنش را تأیید کنید
