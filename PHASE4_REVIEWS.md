# ⭐ فاز 4: سیستم امتیازدهی و نظرات

## 📋 خلاصه فاز 4

سیستم امتیازدهی و نظرات یک پلتفرم کامل برای ارزیابی کاربران، کالاها و خدمات است که شامل امتیازدهی 5 ستاره، نظرات تفصیلی، عکس و ویدیو، و سیستم مدیریت محتوا می‌باشد.

## ✨ ویژگی‌های اصلی

### ⭐ امتیازدهی پیشرفته
- **امتیاز 5 ستاره**: سیستم امتیازدهی استاندارد
- **امتیازدهی دسته‌بندی**: ارتباط، کیفیت، به موقع بودن، حرفه‌ای بودن، ارزش
- **امتیاز میانگین**: محاسبه خودکار میانگین امتیازات
- **توزیع امتیازات**: نمایش توزیع امتیازات 1 تا 5 ستاره

### 📝 نظرات و کامنت‌ها
- **نظرات تفصیلی**: عنوان و محتوای کامل
- **عکس و ویدیو**: آپلود رسانه در نظرات
- **پاسخ به نظرات**: امکان پاسخ فروشنده/کارفرما
- **نظرات تأیید شده**: علامت‌گذاری نظرات معتبر

### 🎯 انواع نظرات
- **خریدار به فروشنده**: نظرات خریداران درباره فروشندگان
- **فروشنده به خریدار**: نظرات فروشندگان درباره خریداران
- **کارفرما به کارگر**: نظرات کارفرمایان درباره کارگران
- **کارگر به کارفرما**: نظرات کارگران درباره کارفرمایان
- **عمومی**: نظرات عمومی

### 🛡️ سیستم مدیریت محتوا
- **تأیید خودکار**: تأیید خودکار نظرات (قابل تغییر)
- **گزارش‌دهی**: گزارش نظرات نامناسب
- **مدیریت**: حذف و ویرایش نظرات
- **فیلتر و جستجو**: فیلتر بر اساس امتیاز و نوع

## 🔧 API Endpoints

### نظرات
```
GET    /api/review                    # دریافت همه نظرات
GET    /api/review/:id                # دریافت نظر خاص
POST   /api/review                    # ایجاد نظر جدید
PUT    /api/review/:id                # ویرایش نظر
DELETE /api/review/:id                # حذف نظر
```

### رأی‌گیری و تعامل
```
POST   /api/review/:id/vote           # رأی مفید/غیرمفید
POST   /api/review/:id/report         # گزارش نظر
POST   /api/review/:id/reply          # پاسخ به نظر
```

### آمار و مدیریت
```
GET    /api/review/user/:id/stats     # آمار نظرات کاربر
GET    /api/review/user/:id           # نظرات کاربر
PUT    /api/review/:id/moderate       # مدیریت نظر (ادمین)
GET    /api/review/reports/pending    # نظرات گزارش شده
```

## 📊 مدل داده

### Review Schema
```javascript
{
  // اطلاعات پایه
  reviewer: ObjectId,              // نویسنده نظر
  reviewedUser: ObjectId,          // کاربر مورد نظر
  product: ObjectId,               // کالای مرتبط
  job: ObjectId,                   // کار مرتبط
  
  // امتیازدهی
  rating: Number,                  // امتیاز 1-5
  categoryRatings: {               // امتیازات دسته‌بندی
    communication: Number,
    quality: Number,
    timeliness: Number,
    professionalism: Number,
    value: Number
  },
  
  // محتوای نظر
  title: String,                   // عنوان نظر
  content: String,                 // محتوای نظر
  photos: [{                       // عکس‌ها
    url: String,
    caption: String,
    uploadedAt: Date
  }],
  videos: [{                       // ویدیوها
    url: String,
    thumbnail: String,
    duration: Number,
    caption: String,
    uploadedAt: Date
  }],
  
  // متادیتا
  reviewType: String,              // نوع نظر
  status: String,                  // وضعیت (در انتظار/تأیید/رد)
  verified: Boolean,               // تأیید شده
  verifiedPurchase: Boolean,       // خرید تأیید شده
  
  // تعامل
  helpful: [{                      // رأی‌های مفید
    user: ObjectId,
    votedAt: Date
  }],
  notHelpful: [{                   // رأی‌های غیرمفید
    user: ObjectId,
    votedAt: Date
  }],
  
  // گزارش‌ها
  reports: [{                      // گزارش‌های نامناسب
    reporter: ObjectId,
    reason: String,
    description: String,
    reportedAt: Date,
    status: String
  }],
  
  // پاسخ‌ها
  replies: [{                      // پاسخ‌ها
    user: ObjectId,
    content: String,
    createdAt: Date,
    updatedAt: Date
  }],
  
  // زمان‌بندی
  createdAt: Date,
  updatedAt: Date,
  deletedAt: Date                  // حذف نرم
}
```

## 🚀 نحوه استفاده

### 1. دسترسی به نظرات
- از صفحه پروفایل روی دکمه "نظرات" کلیک کنید
- یا مستقیماً به `/reviews` بروید

### 2. نوشتن نظر
- روی دکمه "نظر بنویس" کلیک کنید
- امتیاز 1-5 ستاره انتخاب کنید
- عنوان و محتوای نظر را وارد کنید
- عکس یا ویدیو اضافه کنید (اختیاری)
- امتیازات دسته‌بندی را وارد کنید

### 3. مدیریت نظرات
- نظرات خود را ویرایش یا حذف کنید
- به نظرات پاسخ دهید
- نظرات نامناسب را گزارش دهید

### 4. رأی‌گیری
- روی نظرات مفید/غیرمفید کلیک کنید
- نظرات با کیفیت بالا را شناسایی کنید
